# 🎬 AI103 UTCC Marketing Strategy

## 📖 Overview

This project was developed for the **AI103 - Creative Thinking and Innovation** course at the University of the Thai Chamber of Commerce (UTCC).

The objective of this project is to create a complete marketing and branding strategy for the **Digital Film and Acting Program (UTCC)** by applying business analysis and marketing frameworks.

---

## 🎯 Project Objectives

- Analyze the current brand position
- Identify target audiences
- Develop branding strategies
- Increase brand awareness
- Improve student recruitment
- Create a digital marketing plan

---

## 📚 Frameworks Used

- SWOT Analysis
- Business Model Canvas (BMC)
- STP Marketing
- Persona
- Customer Journey Mapping
- Ansoff Matrix
- Value Proposition
- Content Marketing Strategy
